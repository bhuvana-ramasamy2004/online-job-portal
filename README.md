# Online Job Portal

## Project Description
The Online Job Portal is a web application developed to connect job seekers and employers. Users can register, log in, view job listings, post jobs, and apply for available opportunities through a simple and user-friendly interface.

## Features
- User Registration and Login
- View Available Job Listings
- Add and Manage Job Posts
- Apply for Jobs
- View Job Applications
- User-Friendly Interface

## Technologies Used
- JavaScript
- React.js
- Node.js
- Express.js
- mongodb

## Project Status
This project was developed as an academic project to demonstrate full-stack web development concepts.
